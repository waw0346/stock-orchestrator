# 한국 주식 분석 에이전트 — 설정 가이드

## 준비물
- Claude.ai 계정 (Pro 이상 권장 — Project 기능 사용)
- PlayMCP 연결 (DART 데이터용)

## 설치 단계

### Step 1. PlayMCP 커넥터 연결
1. Claude.ai 좌측 하단 **Settings → Connectors**로 이동
2. **PlayMCP** 검색 후 **Connect** 클릭
3. 인증 완료 후 활성화 확인

> PlayMCP 안에 한국 DART(전자공시시스템) 데이터를 가져오는 `opendart-*` 도구들이 들어 있습니다.
> 재무제표, 공시, 주주현황 등 한국 주식 분석의 핵심 원천 데이터입니다.

### Step 2. 새 Project 생성
1. Claude.ai에서 **Projects → Create Project**
2. 이름: 예) "한국 주식 분석 에이전트"
3. Description: "KOSPI/KOSDAQ 종목 분석 및 투자 아이디어 제공"

### Step 3. Custom Instructions 설정
1. 생성한 Project의 **Custom instructions** 섹션 열기
2. `01_project_instructions.md` 파일 내용 전체 복사 → 붙여넣기
3. 저장

### Step 4. (선택) 지식 파일 추가
필요하면 다음 자료들을 Project Knowledge에 업로드해 깊이를 더할 수 있습니다:
- 본인의 투자 원칙 / 종목 선정 기준 문서
- 관심 산업 리포트
- 보유 종목 리스트 (포트폴리오 추적용)

### Step 5. 웹 검색 활성화 확인
Project 안 채팅창에서 우측의 **Tools** 메뉴에서 **Web search**가 켜져 있는지 확인

## 사용 예시

Project 안에서 이런 식으로 질문하세요:

```
삼성전자 분석해줘
```

```
NAVER vs 카카오 비교 분석해줘. 최근 3년 재무 흐름 중심으로.
```

```
LG에너지솔루션 최근 공시랑 최대주주 변동 있는지 확인해줘.
```

```
배당주 중에 ROE 10% 이상이고 배당수익률 4% 넘는 종목 후보 찾아줘.
```

에이전트는 자동으로:
1. opendart로 회사 식별 및 재무 데이터 조회
2. 웹 검색으로 시세/뉴스 보강
3. 정해진 프레임워크대로 정리해서 답변

---

## 트러블슈팅

**"기업을 찾을 수 없다"고 나올 때**
→ 회사명이 정확한지 확인 (예: "삼성전자우" 우선주 vs "삼성전자" 보통주)
→ 종목코드를 직접 알려주면 더 정확

**재무 데이터가 오래된 것 같을 때**
→ DART는 분기·연간 공시 기준이라 최신 분기 미공시 시점에는 직전 분기 자료가 최신일 수 있음
→ 실시간 정보는 웹 검색 결과를 참조

**원하는 분석 깊이로 안 나올 때**
→ 질문에 "재무 중심으로" / "정성적 분석 중심으로" / "리스크 위주로" 같이 방향을 명시
→ 또는 Project Custom Instructions에서 강조하고 싶은 부분 추가

## 다음 단계 아이디어

에이전트가 익숙해지면 확장해볼 수 있는 방향:

1. **포트폴리오 추적기** — 보유 종목 리스트 파일을 Knowledge에 넣고 정기 점검 요청
2. **백테스트 분석** — 과거 특정 시점 기준 의사결정 복기
3. **워치리스트 모니터** — 관심 종목 리스트의 최근 공시·뉴스 일일 브리핑
4. **산업 비교 분석** — 같은 업종 5~10개 종목 횡단 비교

이런 확장은 Project Custom Instructions에 별도 섹션으로 추가하거나, 별도 Project로 분리해도 됩니다.
